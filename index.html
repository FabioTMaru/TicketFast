A extensão valida seu acesso pelo e-mail usado na compra. Para que tudo funcione corretamente, siga estas 3 regras:

3

Na hora do pagamento, use o mesmo e-mail da sua conta Google. Você não precisa ter conta no Mercado Pago — pode pagar com Pix, cartão ou boleto como visitante. Extraia o arquivo .zip

Clique com o botão direito no arquivo baixado e escolha "Extrair tudo" (Windows) ou dê dois cliques (Mac). Uma pasta será criada com os arquivos da extensão.

No Chrome, digite na barra de endereço e pressione Enter:

chrome://extensions

No canto superior direito da página de extensões, ative o botão "Modo do desenvolvedor" (Developer mode).

Clique em "Carregar sem compactação" (Load unpacked) e selecione a pasta que você extraiu no passo 1. A extensão será instalada automaticamente.

Clique no ícone de quebra-cabeça (🧩) na barra do Chrome e fixe o "TicketFast" para fácil acesso.

Ative com seu e-mail

Abra a sidebar

Vá para o Ticketmaster

Acesse o evento no Ticketmaster e chegue na tela de preenchimento dos dados pessoais.

4

Preencha com 1 clique

Na sidebar, clique no botão "⚡ Preencher" da pessoa desejada. Todos os campos serão preenchidos automaticamente. Atalhos rápidos também disponíveis:

Ctrl + Shift + 1 → Pessoa 1   |   Ctrl + Shift + 2 → Pessoa 2
5

Dados de cartão: copie e cole

Para os dados do cartão de crédito, use o botão 📋 ao lado de cada campo na sidebar para copiar e colar manualmente na página de pagamento.

Perguntas frequentes

Funciona no Ticketmaster Brasil?

Sim! A extensão foi desenvolvida especificamente para o Ticketmaster Brasil (ticketmaster.com.br) e também funciona no ticketmaster.com internacional.

Quantas pessoas posso cadastrar?

Quantas precisar! A extensão começa com 2 pessoas e você pode adicionar mais clicando em "+ Adicionar pessoa". Não tem limite.

Meus dados ficam seguros?

Totalmente. Todos os dados são salvos localmente no seu navegador usando a API do Chrome Storage. Nenhuma informação é enviada para servidores externos. Nós não temos acesso aos seus dados.

Preciso instalar algo além do Chrome?

Não. Você só precisa do Google Chrome. A extensão é instalada diretamente no navegador, sem programas extras.

Funciona no celular?

Não, extensões do Chrome funcionam apenas no computador (Windows, Mac ou Linux). No celular, o Chrome não suporta extensões.

Como recebo a extensão após pagar?

Assim que o pagamento for confirmado no Mercado Pago, você será redirecionado automaticamente para a página de download. Se usar Pix, é instantâneo.

A extensão preenche o cartão de crédito também?

Os campos de cartão ficam em uma área protegida do site. Por isso, a extensão oferece botões de cópia rápida (📋) para você copiar cada dado e colar manualmente — é rápido e seguro.

Posso usar para quantos eventos quiser?

Sim! Uma vez instalada, a extensão funciona para todos os eventos do Ticketmaster, sem limite de uso.
