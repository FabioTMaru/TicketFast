Política de Privacidade — TicketFast

Última atualização: 09 de abril de 2026

1. Introdução

O TicketFast é uma extensão para Google Chrome desenvolvida pela FTM Development. Esta política descreve como coletamos, usamos e protegemos as informações dos usuários.

2. Dados coletados

O TicketFast coleta e armazena os seguintes dados, fornecidos voluntariamente pelo próprio usuário:

  • Dados pessoais para preenchimento: nome, sobrenome, CPF, data de nascimento, CIE, validade do CIE, instituição de ensino, cidade, estado, curso e código de acesso (BTS Army ou similar).
  • Email: utilizado exclusivamente para verificação de licença (confirmação de pagamento).
  • Identificador do dispositivo: um código gerado aleatoriamente para vincular a licença a um único computador.

A extensão não coleta histórico de navegação, cookies, senhas, dados de cartão de crédito armazenados no navegador, nem qualquer informação de outras abas ou sites.

3. Armazenamento dos dados

Os dados pessoais para preenchimento (nome, CPF, etc.) são armazenados localmente no navegador do usuário através da API Chrome Storage. Esses dados nunca são enviados para servidores externos.

Os dados de cartão de crédito inseridos na extensão são temporários e apagados automaticamente ao fechar a extensão. Eles não são salvos em nenhum local.

O email e o identificador do dispositivo são armazenados em um banco de dados seguro (Supabase) exclusivamente para fins de verificação de licença.

4. Uso dos dados

Os dados coletados são utilizados exclusivamente para:

  • Preencher automaticamente formulários no site do Ticketmaster conforme solicitado pelo usuário.
  • Verificar se o usuário possui uma licença válida (pagamento confirmado).
  • Vincular a licença a um dispositivo para evitar uso não autorizado.

5. Compartilhamento de dados

Não vendemos, transferimos ou compartilhamos dados dos usuários com terceiros. Nenhuma informação pessoal é enviada para fins de publicidade, análise ou qualquer outro propósito que não seja o funcionamento da extensão.

6. Segurança

Adotamos as seguintes medidas de segurança:

  • Dados pessoais armazenados localmente no navegador (não trafegam pela internet).
  • Comunicação com o servidor de verificação de licença via HTTPS (criptografia TLS).
  • Banco de dados com criptografia em repouso (AES-256).
  • Dados de cartão de crédito nunca são persistidos — são apagados ao fechar a extensão.

7. Permissões da extensão

O TicketFast solicita as seguintes permissões do Chrome, cada uma com sua justificativa:

  • storage: salvar dados do usuário localmente no navegador.
  • activeTab: acessar a aba ativa para preencher formulários do Ticketmaster.
  • scripting: injetar o script de preenchimento automático na página do Ticketmaster.
  • sidePanel: exibir a interface da extensão como painel lateral.
  • tabs: identificar a aba correta do Ticketmaster para preenchimento.
  • identity / identity.email: detectar o email da conta Google para facilitar a verificação de licença.
  • Host permissions (ticketmaster.com, ticketmaster.com.br): acessar os formulários do Ticketmaster para preenchimento automático.
  • Host permissions (supabase.co): verificar a licença do usuário no servidor.

8. Direitos do usuário

O usuário pode, a qualquer momento:

  • Visualizar e editar seus dados diretamente na extensão.
  • Excluir todos os seus dados removendo a extensão do Chrome.
  • Solicitar a exclusão dos dados de licença (email e device_id) entrando em contato conosco.
  • Deslogar da extensão clicando no botão "Sair".

9. Código remoto

O TicketFast não utiliza código remoto. Todo o código da extensão está incluído no pacote instalado. A única comunicação externa é uma chamada de API (fetch) ao servidor Supabase para verificação de licença.

10. Menores de idade

O TicketFast não é direcionado a menores de 13 anos. Não coletamos intencionalmente dados de crianças.

11. Alterações nesta política

Podemos atualizar esta política periodicamente. Alterações significativas serão comunicadas através do site oficial.
